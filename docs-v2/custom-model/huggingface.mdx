---
title: HuggingFace
description: You can bring your Hugging face models as tables to mindsDB
---

Here is an example:

```sql
CREATE PREDICTOR mindsdb.byom_mlflow
PREDICT `1`  -- `1` is the target column name
USING
    format='huggingface',
    APITOKEN='yourapitoken'
    data_dtype={"0": "integer", "1": "integer"};
```
