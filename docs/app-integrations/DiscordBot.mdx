2. Select the server where you want to install the discord bot

## Language Translation Bot :
* Open Discord and send a message containing "!EnglishToFrench " and then pass the word in English which you want to convert into the French language, to the server where the bot is located.

![image](https://user-images.githubusercontent.com/26898623/235345719-9f26776b-de59-4722-a658-ed7b23e91d21.png)
